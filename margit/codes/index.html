Instructions

  1. Drop your .csv file(s) anywhere on the page or use the Choose Files button
  2. Press Process Files button
  3. Click on the files you wish to download

    Rules for the input files

  • Must be a valid .csv file
  • Must use the "," (comma) character as a delimiter
  • Must have a header row with at least the following columns
    • Description
    • Amount
    • Account
